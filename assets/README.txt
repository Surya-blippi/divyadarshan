DivyaDarshan — sample images for the "See The Magic" showcase
=============================================================

The homepage showcase demonstrates the pipeline:
    Your Photo   +   Chosen Temple   =   Divine Portrait

To display YOUR real example (the 3 images you have), save them in THIS
folder with these exact filenames:

    1. sample-you.jpg      →  the plain photo of the person ("You")
    2. sample-result.jpg   →  the final edited portrait (person at Kashi)

The middle "temple" image loads automatically from Wikimedia (Kashi
Vishwanath), so you only need the two files above.

Notes
-----
- Use .jpg files (JPEG). Portrait orientation (roughly 4:5) looks best.
- If these files are missing, the site shows illustrative placeholder
  graphics instead — nothing breaks.
- Want to feature a different temple? Update the middle <img> src in
  index.html (search for "2 · Chosen Temple").
